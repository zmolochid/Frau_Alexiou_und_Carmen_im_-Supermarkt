# Im Supermarkt – Διαδραστικές ασκήσεις

Έτοιμο στατικό υλικό HTML για ανάρτηση σε GitHub και προβολή μέσω GitHub Pages.

## Περιεχόμενα
- `index.html` – η δραστηριότητα
- `.nojekyll` – ώστε το GitHub Pages να σερβίρει το site χωρίς Jekyll επεξεργασία

## Πώς να το ανεβάσεις
1. Δημιούργησε νέο repository στο GitHub.
2. Άνοιξε το repository και επίλεξε **Add file > Upload files**.
3. Ανέβασε τα αρχεία του φακέλου αυτού.
4. Πήγαινε **Settings > Pages**.
5. Στο **Build and deployment**, επίλεξε:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
6. Πάτησε **Save**.
7. Μετά από λίγο, το υλικό θα ανοίγει από το link του GitHub Pages.

## Σημείωση
Το αρχείο είναι αυτόνομο και λειτουργεί χωρίς εξωτερικά assets ή σύνδεση στο ίντερνετ.
